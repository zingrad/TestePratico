# Projeto de Catálogo de Produtos

Este é um projeto de catálogo de produtos desenvolvido usando React.

## Instalação

Para executar este projeto localmente, siga estas etapas:

1. Clone o repositório para o seu computador:

    ```bash
    git clone https://github.com/zingrad/TestePratico.git
    ```

2. No terminal, navegue até o diretório do projeto:

    ```bash
    cd TestePratico
    ```

3. Instale as dependências do projeto:

    ```bash
    npm install
    ```

## Executando o Projeto

Este projeto usa o JSON Server para fornecer uma API de produtos. Para iniciá-lo, abra dois terminais:

1. No primeiro terminal, inicie o JSON Server:

    ```bash
    npm run server
    ```

2. No segundo terminal, inicie o aplicativo React:

    ```bash
    npm run dev
    ```

Isso iniciará o projeto e abrirá o navegador padrão com a aplicação em execução. Agora você pode visualizar e interagir com o catálogo de produtos.
